### npm init 
初始化

### 引入本地webpack
npm i webpack webpack-cli -D

### build 
 
 使用webpack打包

 ### Ts loader

 npm install ts-loader typescript -D

 
 ### tsconfig.json
 初始化tsconfig
 tsc --init

 ### 搭建本地服务

 npm install webpack-dev-server -D

### 配置 config,json
#### 打包webpack
"build" : "webpack"
#### 运行服务
"serve": "webpack serve"
